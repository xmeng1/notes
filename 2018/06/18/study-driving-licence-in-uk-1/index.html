英国学车记第一课

发表于 2018-06-19 | 分类于 生活 |

摘要：开始在英国学习驾照，作为完全新手，很多概念和操作都需要不断熟悉，所以以此博文记录学习过程，一方面加强记忆，有助于快速通过考试，另一方面也分享出来，供大家参考。

概述

本人学习的手动挡的车（Manual Car），自动挡应该除了离合操作外，其他的应该是一样的。

这里不对驾校的选择做评价，一般情况我们最终都和私人教练学习，即使是联系驾校，最终也是和教练联系，所以如果你要开始寻找教练，如果没有熟人推荐，那么大多数情况都是首先选择一家驾校，然后查看一下其价格（大多数官网会报），然后打电话咨询驾校，驾校会根据他们的记录安排空余教练，并预约第一节课。

这里有些驾校是支持在线付款的，但是可能会收取手续费，所以一般情况都是在第一节课首次和教练见面，当面根据预约的课程时间付款。

目前学车的价格大概每10个小时240-260镑，这个需要根据不同驾校来定，但价格不会偏差太多，这里仅供参考。

下面进入正题， 记录一下我的学习过程。

学前准备

这里是自己在网上查找一些自学视频和文章，主要目的是了解

  1. 手动档汽车的原理
  2. 初学注意事项

提前和教练确认事项

和教练首先见面后，首先需要确认几件事情

  1. 查看教练车是否是预定驾校的，这个大部分不会有问题。
  2. 教练会首先查看你的驾照（临时驾照）
  3. 教练会检查你的视力，一般是让你读出某个远处车的车牌
  4. 和教练确定未来课程的时间，以及汇合地点
  5. 和教练确定每次课程学完后，你前往的地点，比如公司或学校
  6. 教练会简单的询问你目前的对于驾驶的掌握程度，如实回答

当这些确认完成之后，一般就会开始学习驾驶，这里由于我是完全新手，所以后面的介绍都是基于新手，所以有可能对于其他水平的人情况不同。

教练驾驶汽车来到他预先选定一篇相对比较空闲的区域来练车。我练车的位置位于一个区中，整体类似于一个正方形，同时环形路线包含上坡下坡，公交站，T型路口等等，应该是教练提前针对课程选择的地点。

基础内容

座椅调节

首先和教练交换位置，坐入驾驶位置，教练坐到副驾驶。

首先教练会指导调整座椅位置，这里主要有两点需要调节，座椅前后位置和靠背前后位置

  1. 通过踩踏离合器到底，来调节座椅前后位置，保证能够很舒适的踩踏离合器完全到底。
  2. 通过双手握住方向盘，来调节座椅后背位置，保证能够很舒适的操作方向盘。

视镜调节

下面开始确认后视镜和左右侧视镜的位置，

  1. 侧后视镜，教练将手放到左侧车门门把手附件，确认能够通过侧视镜看到大概区域，左右侧后视镜原理相同
  2. 中后视镜，确保通过镜子能够看到正中央区域

上路

起步学习

教练简单介绍了离合器和变速器之后，就直接开始起步了，这里一开始有点慌，感觉有点太快，但是走起来之后就感觉没那么慌了。

教练教的起步要领

  1. 确定手刹处于拉起状态，保证汽车静止
  2. 首先左脚完全踩下离合器，并且挂一档
  3. 然后缓慢抬起离合器，直到感觉到 biting point （结合点）
  4. 左脚稳住在 biting point
  5. 开始执行 Observe 操作，即做左后方-> 左后视镜 -> 后视镜 -> 右后视镜 -> 右后方盲点，依次检查是否安全，适合起步，如果不适合，需要等待。
  6. 如果需要提示其他行人车辆，需要打右转向灯，否则可以不需要打
  7. 放下手刹，开始慢慢抬起离合器
  8. 慢慢踩油门提供动力
  9. 方向盘控制前进，如果之前打灯了，需要恢复

总的来说关键的几步

  1. 踩离合一般都会配置换挡
  2. biting point 一开始需要缓慢的抬脚
  3. biting point 找到后，就要观察是否安全
  4. 启动需要放下手刹，并抬起离合器，再踩油门

错误：

  1. 忘记换挡
  2. 忘记放手刹
  3. 到达biting point 后，离合需要hold 住，不能踩下去，也不能松开。

停车学习

平稳的开了一段时间后，教练开始教如何停车。

首先需要确定停车位置是否适合停车，比如离公交站和T路口都至少保持10m的距离等。

一旦准备停车，并且确认停车位置适合停车，按照以下步骤进行

  1. 首先打左转向灯，通知其他人自己的目的是停车
  2. 观察左侧后视镜，确认左侧适合停车，没有行人自行车等。
  3. 抬起油门，将脚放到刹车上，轻轻地点一下刹车
  4. 这时候完全踩下离合器
  5. 然后缓慢的踩下刹车，直到汽车缓慢减速到静止
  6. 等到静止后，拉起手刹
  7. 切换档位为空挡，这里可以通过左右摇晃换挡杆，如果能够自由晃动，说明停到了空挡
  8. 释放离合器和刹车，完成停车

这里关键在于停车的时候，需要一下完全踩下离合器，但是刹车一定要缓慢踩下（这里应该是非紧急停车），这样在停车过程中就不会有较大幅度的颠簸。我在学习的时候，一开始左右脚不太协调，踩刹闸时总是跟随踩离合用力过快，这点需要注意。

行车过程换挡

在行驶过程中，教练会问这时候汽车行驶会有声音（20mph左右），为什么？因为我们起步挂一档，而速度上来之后，一档不适合高速行驶了。

这里如果熟悉了手动变速箱原理后，就容易理解这个问题，

简单来说，档位越低越适合低速运行，因为档位越低，输出的速度所占输入速度越少，这样如果我们试图使用低档位去开高速，发动机就需要更快的输入，也就是转速更快，从而引起声音变化。所以我们往往根据汽车转速来判断是否需要换挡，2000转左右就要换挡，不能让发动机转速过快。

网上有很多介绍换挡时机的经验贴，比如手动挡汽车换挡的标准是什么。当然第一节课主要是学习换挡，还没进入自行判断换挡，教练提示合适适合换挡，我就开始准备换挡

  1. 放开油门
  2. 左脚完全踩下离合器
  3. 切换档位，从一档换到二挡
  4. 松开离合器
  5. 踩油门继续前进

这里主要注意一个问题，左手换挡的动作，教练称之为 Palming method （大概就是手掌操作的技巧）。

比如在从一档换到二挡，教练车左侧上下分别是一档和二挡，这时候应该手掌先从右侧按住换挡杆，然后用手指勾住换挡杆向下拉。为什么这样操作呢？

我一开始直接用手握住换挡杆，然后向下拉，这样做回带来一个问题，当我向下拉的时候，当换挡杆经过空挡中间槽的时候是可以活动的，如果我们没有注意，有可能这时候换挡杆弹到空挡中间，这时候如果直接向下拉，就会换到4档。当然这个问题可以通过注意来避免，但是需要分散精力。

所以教练多次向我提示在换挡中，注意手掌的使用。

这里有个视频介绍这个 Palming Method: How to change gear in car (Palming method)

这里主要是手掌的朝向的问题

  1. 所有目标为 1,2 档的换挡，手掌都应该背对自己
  2. 其他目标，手掌都应该面向自己
  3. 换挡杆到达空挡中间槽会自动弹会空挡，所以操作要放松

上坡启动

这是本课最难的内容，需要左右脚配置。

坡起的主要问题是，在放开手刹的一刻，汽车必须要有动力，否则就会向后滑。这里的操作步骤

  1. 确定手刹处于拉起状态，保证汽车静止
  2. 首先左脚完全踩下离合器，并且挂一档
  3. 然后缓慢抬起离合器，直到感觉到 biting point （结合点），同时轻轻踩下油门，这时候应该能够感觉车要向前走。转速达到 1500 左右。
  4. 左脚稳住在 biting point
  5. 开始执行 Observe 操作，即做左后方-> 左后视镜 -> 后视镜 -> 右后视镜 -> 右后方盲点，依次检查是否安全，适合起步，如果不适合，需要等待。
  6. 如果需要提示其他行人车辆，需要打右转向灯，否则可以不需要打
  7. 放下手刹，biting point 需要保持更长的时间，慢慢抬起离合器，同时慢慢踩下油门
  8. 继续踩油门提供动力
  9. 方向盘控制前进，如果之前打灯了，需要恢复

网上也有很多视频，比如这个： How to do Hill Starts easily in a manual/stick shift car - learning to drive tips

其他问题

教练在教学过程中可能会问问题

  1. 比如故意停车听到一个路口附近，问你我们停车是否合适，这时候需要回忆理论考试中关于停车的要求，离路口，红绿灯，公交车站等至少10米的距离
  2. 停车的时候我们需要首先检查什么？左后视镜，确保没有行人和其他道路使用者。

停车踩离合的原理

如果我们仅仅是减速，只需要踩刹车，但是如果我们需要刹车到停止，我们就需要踩下离合，为什么呢？

因为我们知道，如果不是空挡，离合在不踩下的情况下，发动机是和输出轴连接的，这样如果汽车静止了，发动机也必然停止转动了，这样也就表示发动机熄火了。所以我们在刹车停车时需要踩下离合器，这样车子在停下来时，输出轴和发动机分离，发动机不会熄火，然后再拉上手刹切换到空挡，再放开离合和刹车，这时候由于是空挡，汽车和发动机没有连接，发动机仍然不会熄火，这样我们可以快速了开始下一次启动。

总结

目前第一课的1个小时的课程就总结到这里，下节课应该是进行路口转向练习。
