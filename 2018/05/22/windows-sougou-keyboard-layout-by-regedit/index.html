使用注册表搜狗输入法Windows修改键盘布局

发表于 2018-05-23 | 分类于 知识积累 |

摘要：搜狗 Mac 版已经支持键盘布局的设置，但是 Windows 一直没有支持，所以只能另辟蹊径，修改注册表来实现。

实际上原理很简单，只需要修改注册表中 中文简体对应的键盘 DLL就可以了

首先打开注册表，Win 按键，然后输入： RegEdit

找到对应的键修改即可

路径： Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layouts\00000804

修改 Layout File ： KBDUS.DLL 为指定的配置，比如修改为英国键盘，即为 KBDUK.DLL。其他国家键盘布局可以通过查看其他的键值来查看，比如 00000807 是 Swiss German，就是 KBDSG.DLL。
